# Bass Diffusion Model and Barabasi-Albert Network Analysis
*Моделирование диффузии цифрового продукта на сети агентов*

---

## EN

Modeling the diffusion of a digital product on an agent network using the Bass model and Barabasi-Albert network structure. Three market penetration scenarios with cumulative adoption dynamics.

**Tools:** Python, NumPy, Matplotlib

## Run
```bash
python diffusion_model.py
```
